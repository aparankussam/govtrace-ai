# GoVTraceAI

**Check before you send.**

GoVTraceAI is a simple AI safety tool that helps users evaluate whether a prompt or AI-generated output is safe before use.

## What it does
- Detects sensitive data (PII/PHI)
- Flags risky prompts and prompt injection patterns
- Identifies unsafe or hallucination-prone AI outputs
- Returns a simple result:
  - SAFE
  - WARNING
  - BLOCK

## Vision
Before you send anything to AI, check if it is safe.

## MVP Scope
- One screen
- One action: **Check Risk**
- One result: SAFE / WARNING / BLOCK
- Lightweight heuristic-based analysis

## Disclaimer
This tool provides automated risk screening and does not constitute legal or regulatory advice.
