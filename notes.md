Product: GoVTraceAI
Company: Gobots LLC
Tagline: Check before you send.

Vision:
Before a user sends anything to AI, GoVTraceAI checks whether it is safe.

MVP:
- one screen
- one action
- one result
- SAFE / WARNING / BLOCK

Core checks:
- PII/PHI exposure
- prompt injection patterns
- risky/hallucination-prone content
- basic policy/compliance-style issues

Platform:
- web-first
- works on MacBook and iPhone
- not a native app for v1
